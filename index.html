# Ontopic prize

For the NOI Hachathon SFScon edition 2019, Ontopic is proposing you to develop an Amazon Alexa skill using the Virtual Knowledge Graph (VKG) of the South-Tyrolean Open Data Hub (ODH) (slides).

# Price criteria

  • Use data from the ODH SPARQL endpoint for constructing Alexa responses
  • Alexa responses should appear as natural and as relevant as possible

# How to proceed

In the website, we provide you some suggestions on how to:

  • Setup the environment on your machine
  • Develop an Alexa skill
  • Query the VKG using some pre-defined SPARQL queries and by looking at its schema
